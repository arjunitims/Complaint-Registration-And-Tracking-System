# Complaint-Registration-And-Tracking-System
MCA project
## Description
A web-based complaint management system using Flask and SQLite.

## Requirements
- Python 3.11.9(should install)


## How to Run
1. Install Python
2. Install required packages
3. ie; pip install flask
      pip install numpy
      pip install scikit-learn
      pip install pickle-mixin
5. Run:
   python app.py
6. Open browser:
   http://127.0.0.1:5000
